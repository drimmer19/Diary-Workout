# Дневник тренировок

Описание функционала приложения:

Интерактивная карта с библиотекой Leaflet: отображение текущей позиции пользователя и установка маркера по клику.

Добавление тренировок двух типов: бег (running) и велосипед (cycling). Форма переключается между полями cadence и elevation.

Проверка вводимых данных: числовые значения и положительные числа, с показом ошибки при неверном вводе.

Объектно‑ориентированная модель: классы Workout, Running, Cycling — вычисление темпа (pace) и скорости (speed), автоматическое формирование описания с датой.

Рендер маркеров с попапами (эмодзи и описание) и бокового списка тренировок с деталями (дистанция, время, темп/скорость, шаги/подъём).

Клик по элементу списка центрирует карту на соответствующей тренировке с плавной анимацией.

Сохранение тренировок в localStorage и восстановление при перезагрузке страницы.

UX: автоподстановка фокуса на поле, очистка формы и её скрытие после отправки, возможность сброса сохранённых данных.
